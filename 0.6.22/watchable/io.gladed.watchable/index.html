Watchable
Link copied to clipboard

An object that allows you to watch for changes of type C.

Each watch operation takes a CoroutineScope. Callbacks are delivered using this scope's context, and stop automatically when this scope cancels or completes.

Each watch operation also returns a Watcher which may be used to independently cancel or join the watch operation.
